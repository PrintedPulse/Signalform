# SignalForm

### Pure Hi-Fi. No Apps. Just Sound.

**by Eric Brunner / [Printed Pulse](https://makerworld.com/en/@ericbrunner)**

-----

SignalForm was supposed to be a crowdfunded hi-fi Bluetooth boombox. It got featured by Bambu Lab. Then it didn’t fund.

That’s the honest version.

Instead of letting it sit in my files, I’m putting it out here for anyone who wants to build it, improve it, or take it somewhere I didn’t. The only ask: put my name on it.

-----

## What It Is

A 3D-printed stereo boombox built around one idea — simple electricity. No apps. No Wi-Fi. No cloud pairing. Just clean Bluetooth audio, tactile controls, and a rechargeable battery you can take anywhere.

**Specs:**

- 2× Dayton Audio DMA105-8 4” Full-Range Aluminum Cone Drivers (8Ω)
- 2× Dayton Audio ND105-PR 4” Passive Radiators (natural bass extension)
- Dayton Audio KABD-250 2×50W Amp Board with DSP and Bluetooth 5.0 aptX HD
- Dayton Audio KABD-SPF Function Cables
- Dayton Audio LBB-5Sv2 5×18650 Battery Charger Board 21V
- Dayton Audio LBB-5CL DC Charging + Power Cable Kit
- 5× 18650 3.7V Li-Ion batteries
- 12mm stainless steel latching push button (power)
- Panel-mount USB-C charging port
- Printed in PETG

**Total component cost: ~$160 in parts**

## Print Settings

- **Material:** PETG (required — heat tolerance for electronics bay)
- **Layer height:** 0.2mm
- **Infill:** 30% minimum for speaker baffles, 15% elsewhere
- **Supports:** Required for driver mounting flanges
- **Hardware needed:** M3 heat set inserts, M3 screws, 5-minute epoxy, clear silicone, CA glue, hot glue
